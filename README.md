# Start app
### 1. Create venv
> Linux/Mac
```bash
python -m venv venv
```

>Windows
```bash
py -m venv venv
```


### 2. Activate venv
> Linux/Mac
```bash
source venv/bin/activate
```

>Windows
```shell
venv\Scripts\activate
```

  
### 3. Install requirements
```bash
pip install -r requirements.txt
```


### 4. Start server
```bash
python manage.py runserver
```
